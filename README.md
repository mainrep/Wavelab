# Wavelab [Underdevelopment]
Wavelab is an interactive wave simulator, available to make learning Physics easier using an interactive simulator, detailed explination, and more.

## Goals
- An interactive wave simulator that may be used by students to grasp wave mechanics with relative ease.
- A frontend interface which explains each concept in a thorough manner.
- An interactive assistant to solve any doubts using an API backend (Gemini).

## Technologies
1. HTML
2. CSS
3. Python
4. AI API
